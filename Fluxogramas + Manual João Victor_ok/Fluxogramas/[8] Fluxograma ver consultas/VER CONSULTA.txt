O fluxograma da Figura () mostra o processo no qual o usuário verá as consultas marcadas em determinado dia, logo o usuário deve informar o dia e o mês desejado para ver se existem consultadas marcadas no dia especificado, caso exista alguma consulta marcada para o dia, o sistema exibirá todos os dados da consulta, caso não haja consulta, o usuário será redirecionado para o menu principal.
